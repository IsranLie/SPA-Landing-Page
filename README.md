### Setup Instructions

To get a local copy of the code, clone it using git:

```
git clone https://github.com/IsranLie/SPA-Landing-Page.git
cd SPA-Landing-Page
```

Install depedencies:

```
npm install
```

Now, you can start a local web server by running:

```
npm run dev
```
